4 ELKS 0.1.0-pre4

Kernel release. This release moves the serial ports to their correct node numbers, rather than occupying nodes reserved for virtual terminals. It also allows the ELKS kernel to report its version number to any program that asks for it. Additionally, with this release, ELKS no longer relies on the Linux kernel source being available at /usr/src/linux for ELKS to compile.

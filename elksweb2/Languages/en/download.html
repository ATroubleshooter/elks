CVS (Concurrent Versioning System)

You can also use CVS to get the latest snapshot of the various ELKS parts. This project's SourceForge CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. The module you wish to check out must be specified as the modulename. When prompted for a password for anonymous simply press the <Enter> key.

    cvs -d:pserver:anonymous@cvs.elks.sf.net:/cvsroot/elks login
cvs -z3 -d:pserver:anonymous@cvs.elks.sf.net:/cvsroot/elks co modulename

The available modules are:

  • elks (The ELKS kernel)
  • elksnet (The ELKS ipv4 network code)
  • elkscmd (Collection of userspace software for ELKS)

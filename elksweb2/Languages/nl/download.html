CVS

Je kunt ook gebruikmaken van CVS om een laatste snapshot te krijgen van de verschillende ELKS onderdelen. Dit project's SourceForge CVS repository kan bekeken worden als anonymous (pserver) CVS met de volgende instructies. The module die je wilt bekijken moet opgegeven worden als modulenaam. Wanneer er om een wachtwoord gevraagd wordt voor anonymous, druk dan gewoon op Enter toets.

cvs -d:pserver:anonymous@cvs.elks.sourceforge.net:/cvsroot/elks login
cvs -z3 -d:pserver:anonymous@cvs.elks.sourceforge.net:/cvsroot/elks co modulenaam

De beschikbare modules zijn:
  • elks (De ELKS kernel)
  • elksnet (De ELKS ipv4 netwerk code)
  • elkscmd (Collectie van userspace programma's voor ELKS)

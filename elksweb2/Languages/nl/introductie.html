Het doel van het ELKS poject is het maken van een Linux kernel voor:

  • 8086 and 80286 klasse IBM Personal Computer compatible systemen,
  • Palmtop Computers, zoals de Psion SiBO (Zestien Bit Organiser),
  • Single board microcomputers,
  • Embedded controller systemen, and
  • Andere oude computers.
De huidige status van het ELKS project op de verschillende gesupporte architecturen is:
  1. ELKS draait nu op IBM Personal Computer compatible systemen. Op deze architectuur, heeft de ELKS kernel ongeveer 200K nodig. Een volledig draaiend systeem heeft dan tussen de 400K en 512K aan RAM nodig.
  2. ELKS draait op de Psion SiBO (Zestien Bit Organiser) palmtop computer.
  3. Er is geen versie bekend van ELKS voor een andere architectuur.
Het Linux operating system wordt gebruikt als ontwikkel platform voor ELKS.

De ELKS programma code bevat een klaar om te bouwen kernel en root disk images om de eerste stappen richting ELKS makkelijker te maken.
